# TailwindCss-LandingPag
This is a travel page build in HTML and tailwindcss. With mobile design & dark mode. 

## Local install 
~~~
git clone https://github.com/Thefederico/TailwindCss-LandingPage.git

cd TailwindCss-LandingPage

yarn
~~~

### Run project
~~~
yarn tw:dev
~~~

### BUild project
~~~
yarn tw:build
~~~
